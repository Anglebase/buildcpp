# PyCMake

![MIT-LICENSE](https://img.shields.io/github/license/Anglebase/PyCMake)
![PyPI](https://img.shields.io/pypi/v/pycmake)

PyCMake is an innovative build system that combines Python's expressive power with the reliability of CMake. No longer need to worry about complex CMake syntax - manage your C++projects with clear and maintainable Python code.
