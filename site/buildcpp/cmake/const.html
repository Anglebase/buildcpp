class Scope(Enum):
    """
    Scope of action.

    Some functions expect to accept an enumeration value indicating their scope of action.
    - `PUBLIC` means that the item is visible to itself and all targets that depend on it.
    - `PRIVATE` means that the item is only visible to itself.
    - `INTERFACE` A means that the item is only visible to its dependencies.
    """
    PUBLIC = "PUBLIC"
    PRIVATE = "PRIVATE"
    INTERFACE = "INTERFACE"

Scope of action.

Some functions expect to accept an enumeration value indicating their scope of action. - PUBLIC means that the item is visible to itself and all targets that depend on it. - PRIVATE means that the item is only visible to itself. - INTERFACE A means that the item is only visible to its dependencies.
